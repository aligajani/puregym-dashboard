# Legacy — Flask (Python)

Original Python/Flask implementation of the PureGym dashboard.

Superseded by the Next.js app at the repo root.

## Run

```bash
python -m venv venv
source venv/bin/activate
pip install flask puregym-attendance
python app.py
# → http://localhost:8080
```

Credentials are hardcoded in `app.py`. Not intended for public deployment.
